# megarover_change_speed_angular
## How to Use
1. Launch ```roscore```.
2. Run ```rosrun megarover_change_speed_angular megarover_change_speed_angular``` in another terminal.
    - Option
        - -s : Linear Speed [m/s] (default : 0.0)
        - -a : Angular velocity [rad/s] (default : 0.0)
        - -h : Print Help
